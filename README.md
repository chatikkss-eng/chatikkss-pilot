# Chatikkss Pilot 🚀
**Умный AI-ассистент типа Джарвиса**

![Chatikkss Hero](https://i.ibb.co/HLLJMKjk/Untitled-3.png)

Chatikkss Pilot это проект, рожденный из идеи объединить мощь современных нейросетей с полным контролем над операционной системой. Это не просто чат, это твой цифровой пилот, который помогает управлять рутиной, не отвлекаясь от важных дел.

---

## 🔥 Что умеет Chatikkss?

*    **Интеллект нового поколения:** Внутри живет Mistral Large — модель ии, которая помнит твои предпочтения и контекст разговора.
*    **Голос как инструмент:** Забудь про ввод команд руками. Просто скажи "Открой браузер" или "Найди инфу про Python", и ассистент всё сделает.
*    **Безопасность превыше всего:** Система **Safe Actions** гарантирует, что ассистент не сделает ничего лишнего без твоего подтверждения.
*    **Премиальный UI:** Современный интерфейс с эффектом матового стекла (Glassmorphism), который идеально вписывается в дизайн Windows.
*    **Полная прозрачность:** Это Open Source проект. Ты видишь, как он работает, и можешь быть уверен в приватности своих данных.

---

## 🛠️ Технологии под капотом

*   **Сердце:** Python 3.10+ & Flask
*   **Мозги:** Mistral AI API
*   **Связь:** WebSockets для мгновенного отклика
*   **Голос:** Microsoft Edge-TTS & Google Speech Recognition
*   **Оболочка:** Интерактивное веб-окно (PyWebView)

## 🚀 Как запустить (для тех, кто хочет покопаться в коде)

1. Установите зависимости командой `pip install -r requirements.txt`.
2. Добавьте свой `MISTRAL_API_KEY` в файл `.env`.
3. Запустите основной файл: `python desktop.py`.

*Примечание: Если вы просто хотите пользоваться программой, скачайте готовую сборку в разделе [Releases](https://github.com/chatikkss-eng/chatikkss-pilot/releases).*
